# 半导体设备溅射靶材

> 超高纯金属靶材（铝、钛、钽、铜），用于 PVD（物理气相沉积）设备，将金属薄膜沉积到晶圆表面。占薄膜沉积设备价值量 3-8%。

## 技术壁垒

- **等级**: 高
- **核心要求**: 超高纯度（5N-6N，99.999%-99.9999%）、晶粒均匀、致密性、与背板焊接强度
- **关键工艺**: 超高纯金属提纯 → 熔炼/粉末冶金 → 热等静压 → 精密加工 → 与背板绑定
- **认证周期**: 长（2-3年，需配合 PVD 设备验证）
- **替代难度**: 霍尼韦尔（美国）、东曹（日本）主导，国产化率较高（江丰电子主导）
