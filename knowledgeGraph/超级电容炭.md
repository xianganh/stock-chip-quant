# 超级电容炭

## 定义
超级电容活性炭（电容炭）是超级电容器的核心上游材料，作为电极材料直接决定电容的储能密度、导电性能和循环寿命。高比表面积电容炭具有技术密集、高工艺壁垒特征。

## 产业链位置
```
上游：木质/煤质碳源 → 超级电容炭（本节点）→ 超级电容器 → GB300 AI服务器
```

## 产业格局
- 全球高端市场由日本可乐丽（Kuraray）垄断
- 元力股份（300174）为国内少数实现量产的企业，产能 1000 吨/年
- AI服务器GB300标配超级电容 → 电容炭需求指数级增长
