# 球形硅微粉

## 定义
球形硅微粉是先进封装（HBM、Chiplet等）的关键填充材料，用于EMC（环氧模塑料）和Underfill（底部填充胶）中，降低热膨胀系数、提高导热性。

## 产业链位置
```
硅酸钠 → 白炭黑 → 高纯球形硅微粉 → EMC/Underfill → 先进封装（HBM/Chiplet）
```

## 产业格局
- 全球高端市场由日本电化（Denka）、龙森（Tatsumori）合计垄断 70%+
- 国内高端产品国产化率不足 10%
- 日系扩产周期 2-3 年，2027 年前无新增有效供给
- 全球高端球形硅微粉 2026-2028 CAGR > 40%
