# CPO（光电共封装）

> **CPO（Co-Packaged Optics）** 是将光引擎与交换芯片封装在一起的技术，取代传统可插拔光模块，是AI算力基础设施的关键技术方向。

## 技术定义

| 属性 | 说明 |
|------|------|
| 全称 | Co-Packaged Optics（光电共封装） |
| 核心 | 将光引擎（optical engine）与交换芯片（switch ASIC）封装在同一基板上 |
| 替代对象 | 传统可插拔光模块（pluggable optics） |
| 优势 | 降低功耗、缩短信号路径、提升带宽密度 |
| 产业化阶段 | 2025-2026年早期商用，2027-2028年规模部署 |

- **产业化进度**：2025-2026年客户验证阶段，订单尚未大规模落地
