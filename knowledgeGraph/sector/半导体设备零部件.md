## 产业链结构

```
晶圆制造厂（中芯国际、长江存储、长鑫存储）
  → 设备整机（刻蚀/薄膜/光刻/清洗/量检测）
    → 零部件供应商（本产业链）
      ├─ 机械类：腔体、内衬、基座、冷却板（富创精密、江丰电子）
      ├─ 陶瓷类：静电吸盘、陶瓷腔体（珂玛科技、中瓷电子）
      ├─ 气体类：阀门、管路、气柜（新莱应材、VAT）
      ├─ 靶材：超高纯金属靶（江丰电子）
      ├─ 机电类：机械手、射频电源（新松、华卓精科）
      ├─ 光学类：镜头、光源（奥普光电）
      └─ 散热类：水冷组件（斯瑞新材）
```
