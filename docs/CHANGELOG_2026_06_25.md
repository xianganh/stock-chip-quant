## 🎯 核心方法论沉淀

### 1. 定性优先原则
- **不能单纯纯数据验证**
- 每一个信号必须有定性理论支撑
- Winner 不能作为单一指标

### 2. 反直觉的实战洞察

| 现象 | 解释 |
|------|------|
| 出货时 Winner 接近 100% | 所有人都赚钱 = 主力拉高出货 |
| 见顶时 TP3/Kurtosis 最高 | 高度集中 = 主力已完全控盘 |
| 上涨中 Winner 下降 = 洗盘 | 主力在让获利盘下车 |
| 上涨中 Winner 上升 = 散户追涨 | 反而是出货前兆 |

### 3. 行情类型适配
- **单边上涨**: 任何择时都跑输持有
- **单边下跌**: 信号系统减亏10-15%
- **震荡市**: 信号系统小幅跑赢
- **真实价值**: 控制回撤，不是绝对收益

## 💡 关键经验

1. **回测时间窗口选择很关键** - 在单边上涨行情中测试择时策略会误导
2. **定性理论是不可或缺的** - 纯数据验证容易过拟合
3. **跨股验证是必要的** - 单一股票的结论不可靠
4. **分级机制优于单一阈值** - 预警/高度预警/清仓分级更合理
5. **出货特征是反直觉的** - Winner 接近 100% 才是真正的出货信号
